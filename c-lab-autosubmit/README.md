# 🚀 C-Lab AutoSubmit

사이버보안 전공 1학년 C 프로그래밍 실습을 위한 **공식 실습 및 자동 제출 시스템**입니다. 
실습실 PC에서 안전하고 편리하게 코드를 작성하고 제출할 수 있도록 돕습니다.

## 📖 사용 방법

### 1. 실습 시작하기
1. 수업 시간이 되면 VS Code 우측 하단 상태 표시줄에 파란색 **`▶️ C-Lab 실습 시작`** 버튼이 나타납니다.
2. 버튼을 클릭하면 실습을 진행할 전용 폴더가 자동으로 열립니다.
3. 본인의 **학번(8자리)**과 **이름**을 정확하게 입력해 주세요.
4. 생성이 완료된 `practice_main.c` 파일에 오늘 배운 C 언어 코드를 작성합니다.

### 2. 과제 제출하기
1. 코딩을 모두 마쳤다면 우측 하단의 주황색 **`🚀 C-Lab 최종 제출`** 버튼을 클릭합니다.
2. 경고 팝업이 뜨면 **[예]**를 눌러 제출을 확정합니다.
3. 🎉 **제출 완료 메시지**가 뜨면 정상적으로 서버에 코드가 전송된 것입니다.
4. **주의:** 제출이 완료되면 보안을 위해 작성했던 코드는 즉시 영구 삭제되며, VS Code 화면이 초기화됩니다. 제출 전 코드가 완벽한지 반드시 확인하세요!

---

## ⚠️ 실습 시 주의사항 (필독)

* **상시 저장 및 백업:** 실습이 시작되면 1초 단위로 코드가 자동 저장되며, 작성 중인 코드의 변경 사항(Diff)과 디버깅 과정은 서버로 안전하게 백업됩니다.
* **비정상 종료 시 복구:** 실수로 VS Code 창을 끄거나 컴퓨터가 재부팅 되더라도 당황하지 마세요. 다시 켜면 이전에 입력한 학번과 작성 중이던 코드가 **자동으로 복구**됩니다.
* **부정행위 방지 시스템:** 다른 컴퓨터에서 코드를 복사해 오거나, 실습 시간을 벗어나 비정상적인 방법으로 제출을 시도할 경우 IP 및 Machine ID 교차 검증 시스템에 의해 차단 및 기록됩니다.
* **자동 제출:** 실습 시간이 종료되거나, 본인이 제출하지 않고 자리를 떠나 다음 날 켜질 경우 남아있는 코드는 서버로 강제 자동 제출됩니다.

> 💡 **문의사항:** 시스템 오류나 제출 관련 문제가 발생할 경우 즉시 코딩 트레이너 또는 수업 조교에게 문의해 주세요.
